Literals

In Java, literals are fixed, constant values represented directly in the source code. They are not variables or expressions but rather the raw values themselves that are assigned to variables or used in expressions.

Arrays

Arrays in Java are fundamental data structures used to store a fixed-size sequential collection of elements of the same data type. This means an array can hold multiple values of the same type (e.g., all integers, all strings) under a single variable name.

Identifiers

Identifiers in Java are names used to identify various programming element such as classes, methods, variables, packages, and interfaces. They are essential for uniquely identifying and referencing these components within a Java program.
